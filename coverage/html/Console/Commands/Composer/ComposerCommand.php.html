21 * This command provides direct access to Composer commands within workspace contexts.

27 * options, and behavior of the underlying Composer command. This makes it a flexible
28 * tool for any Composer operation that isn't covered by specialized commands like
29 * require or update.
30 *
31 * Features:
32 * - Run any Composer command in workspace context
33 * - Interactive workspace selection if not specified
34 * - Workspace validation before execution
35 * - Full argument passthrough to Composer
36 * - Support for all Composer flags and options
37 * - Real-time command output streaming
38 * - Automatic working directory management
39 *
40 * Common use cases:
41 * - Show installed packages (composer show)
42 * - Dump autoloader (composer dump-autoload)
43 * - Validate composer.json (composer validate)
44 * - Check for security issues (composer audit)
